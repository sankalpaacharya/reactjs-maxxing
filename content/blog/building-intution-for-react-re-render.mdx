# Sync Data with UI

The role of many frontend frameworks either it's React, Vue, Angular, Selvte or Solid is to simply 
sync your data with the UI. DOM manipulation is quite expensive because your browser has to 
go through the process of repaint, relflow. 

- reflow: calculating where your each elements, images and etc should be
- Repaint: actualling drawing pixels in the screen

Now all these library or the framework are just making 
the ease for the Data syncing with the UI with a fewer DOM manipulation possible.
